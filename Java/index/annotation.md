# 1. 注释

**注释** 是一个在不影响代码功能的同时添加文字的功能，常常被用于解释代码。  
注释内的内容会被编译器忽视。  
当 **注释** 用作动词时，指用注释功能将某段代码变成别编译器忽略的字符。  

## 1.1 单行注释

单行注释用```//```开头，此后字符都被注释。  

```java
System.out.println("a");//输出 a  
//System.out.println("b");  
```

> 输出：  
> a  

## 1.2 多行注释

多行注释用 ```/*``` 开头，以 ```*/``` 结尾，其间的所有字符都被注释  

```java
System.out.println/*输出 a*/("a");  
/*
System.out.println("b");  
*/
```

> 输出：  
> a
