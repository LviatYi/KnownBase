# 5. 水平线

使用 ```<hr/>``` 创建水平线。  
水平线能很好地分割段落，使文档层次鲜明。

[示例 5-1](../example/5-1.html)

 ```HTML
<h6>
    Content 1
</h6>
<p>
    This is paragraph 1.
</p>

<hr/>

<h6>
    Content 2
</h6>
<p>
    This is paragraph 2.
</p>

<hr/>

<h6>
    Content 3
</h6>
<p>
    This is paragraph 3.
</p>
</html>
 ```
