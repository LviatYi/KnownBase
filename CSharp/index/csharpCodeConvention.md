# C# 编码约定

## 文档用语

本文档描述约定时将采用 RFC2119 协议规范。将采用如下词汇描述规范条目的重要程度：

| 词汇                   | 含义                                       |
| ---------------------- | ------------------------------------------ |
| 必须 MUST              | 规范的绝对要求                             |
| 禁止 MUST NOT          | 规范的绝对禁止                             |
| 推荐 RECOMMENDED       | 可能存在正当理由以忽略条目，但必须权衡利害 |
| 不推荐 NOT RECOMMENDED | 可能存在正当理由以忽略条目，但必须权衡利害 |
| 可以 MAY               | 某些场合下可以选用条目                     |

[RFC2119][rfc2119]

## 命名约定

### Pascal case

class, record, or struct **必须** 使用 Pascal case
