## 基本介绍

数据库是程序员必备技能之一。本份 Note 以实际应用为目标，包含数据库理论、设计、应用三大方面的知识。  

同时为了便于参考，Note 还提供了常用函数的代码示例与使用手册。  

## Note 简介

本份 Note 旨在记录学习进度，提供部分参考，部分内容可能不甚严谨。  

类似「每条 Java 语句都需要以半角分号 ```;``` 结尾」这种重要知识点不会在 Note 中被提及，因此不适合新手学习。
